10.5.3. Task 1#

Introduction#

The following task demonstrates scalar and row/column vector manipulations. These include matrix methods and element by element methods. MATLAB was designed to perform matrix operations by default. All mathematical operations involving element-wise multiplication, division, and exponentiation of vectors requires suppression of the default matrix math. This is done using dot notation, or the use of a period before the following symbols:

  • Multiplication: *

  • Division: /

  • Exponentiation: ^

Task Instructions#

  1. Open the MATLAB Template and fill out the header information. Then save the file as 10_1_1_teamnumber.m

  2. In the %% INTIALIZATION section, create the following scalars and vectors:

  • Ascalar = 3

  • Arowvector = [0 1 2 3]

  • Browvector with four elements starting at \(4\) and ending at \(-2\) with increments of \(-2\). Do this by typing Browvector = 4:-2:-2 NOTE: this notation start: increment value: end allows you to create vectors of any size with any starting value, increment size, and ending value.

  • Crowvector with three elements of your choice between \(-5\) and \(5\) inclusive.

  • Acolvector = [0; 1; 2; 3]

    Note

    This is a \(4 \times 1\) column vector, which is different from Arowvector. You could also create Acolvector by computing the transpose of Arowvector by typing Acolvector = Arowvector'

  • {MATLAB}`Bcolvector = [-4; -3; -2; -1]

  1. Open the Task 1 & 2 answer sheet (10_1_teamnumber.pdf). Then compute each of the operations under the %% CALCULATIONS section of your script.

Part A#

Fill in the table with the calculated output and an explanation of what MATLAB did to get the result or why MATLAB cannot perform the operation. NOTE: If there are any calculations that cause errors, comment it out and re-run the code.
i. Calc1 = Arowvector + Arowvector
ii. Calc2 = Arowvector + Browvector
iii. Calc3 = Arowvector + Ascalar
iv. Calc4 = Arowvector – Arowvector
v. Calc5 = Arowvector – Crowvector
vi. Calc6 = Acolvector + Bcolvector
vii. Calc7 = Arowvector + Bcolvector

Part B#

Fill in the table with the calculated output and an explanation of what MATLAB did to get the result or why MATLAB cannot perform the operation. NOTE: If there are any calculations that cause errors, comment it out and re-run the code.
i. Calc8 = Arowvector * Browvector
ii. Calc9 = Arowvector .* Browvector
iii. Calc10 = Arowvector * Ascalar
iv. Calc11 = Arowvector .* Ascalar
v. Calc12 = Arowvector ./ Browvector
vi. Calc 13 = Arowvector ^ Ascalar
vii. Calc14 = Arowvector .^ Ascalar
